# Cosmetics Catalog

Responsive cosmetics catalog section built with clean HTML and CSS.

## Features

- 6 product cards
- Responsive layout
- Hover effects
- Clean typography
- Desktop, tablet and mobile adaptation
- Built without UI libraries

## Technologies

- HTML5
- CSS3
